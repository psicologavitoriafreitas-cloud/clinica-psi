Para usar o sistema, você precisa de uma Client ID do Google. Siga os passos abaixo (5 minutos):

Passo 1
Acesse o Google Cloud Console
Vá em console.cloud.google.com, crie um projeto novo (ex: "Clínica Psi") e ative as APIs: Google Calendar API e Google Drive API.
Passo 2
Crie as credenciais OAuth
Em APIs & Services → Credentials, clique em Create Credentials → OAuth Client ID. Escolha tipo Web application. Em "Authorized JavaScript origins" adicione: null (para arquivo local).

Cole o Client ID abaixo
